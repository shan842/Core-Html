Common HTML Tags

  1. <html> - The root element of an HTML document. All other HTML elements are nested inside it.
  2. <head> - Contains metadata about the HTML document, such as the title, character set, and links to external resources like CSS files.
  3. <title> - Specifies the title of the document, which appears in the browser's title bar or tab.
  4. <body> - Contains the content of the web page, including text, images, links, and other media.
  5. <h1> to <h6> - These tags define headings, with <h1> being the most important and <h6> the least.
  6. <p> - Defines a paragraph of text.
  7. <a> - Creates a hyperlink to another web page or resource. The href attribute specifies the link destination.
  8. <img> - Embeds an image in the web page. The src attribute specifies the image file path.
  9. <ul> - Defines an unordered list, typically with bullet points. It contains <li> tags for each list item.
  10. <ol> - Defines an ordered list, typically with numbered items. It also uses <li> for each list item.
  11. <li> - Represents a list item in either an ordered or unordered list.
  12. <div> - A block-level container used to group elements for styling or scripting purposes.
  13. <span> - An inline container used to apply styles or class/id attributes to a section of text or other inline content.
  14. <form> - Defines an HTML form for user input. It contains elements like <input>, <textarea>, and <button> for form controls.
  15. <input> - Defines an input field in a form, where the user can enter data. It can be of various types like text, password, or checkbox.
  16. <button> - Creates a clickable button. It can be used for submitting forms or triggering actions with JavaScript.
